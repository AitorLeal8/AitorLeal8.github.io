Aplicación de contactos CRUD

Agenda de contactos CRUD (create, replace, update and delete) con PHP y MySQL, hecha a partir del curso de PHP de Antonio Sarosi. La app maneja un registro y un login, una vez el usuario haya creado su cuenta podra iniciar sesion. Un usuario puede añadir contactos a su cuenta, una vez creados puede editarlos y borrarlos.

Scripts en python

Base de datos en ORACLE SQL
